# Netflix Data Analysis

EDA on Netflix dataset (8,807 rows, 12 columns) using Python & Pandas.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Movies vs TV Shows distribution
- Top 10 countries by content
- Content added over years
- Top 10 genres
- Rating distribution
- Movie duration distribution

## Outcome
- Identified content trends and distribution patterns
- Data cleaning using fillna() and null value removal
